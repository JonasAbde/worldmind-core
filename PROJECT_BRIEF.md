# Project Worldmind

## Working title
Plain text WorldMind Engine

## Runtime name
Plain text HermesWorld Core

## Core concept
A living open-world AI simulation game where each NPC is a sandboxed AI agent with:

- memory
- goals
- skills
- tools
- relationships
- economy state
- permissions

Quests and story should emerge from the simulation itself.

## Asset direction
Visual assets are intended to be generated with Higgsfield. The project now includes a ready prompt pack in `assets/higgsfield-prompts.md`.

## Source seed
This workspace was seeded from `worldmind.zip` and copied into this project folder.
