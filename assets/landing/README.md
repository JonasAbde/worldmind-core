# Landing / product assets

Marketing and landing-page art for WorldMind. These files are **not** runtime gameplay sprites.

Runtime paths used by `src/play/assets.js` remain under `assets/hero`, `assets/locations`, `assets/characters`, `assets/ui`, etc.

See `docs/51_ASSET_ART_LOCK_LANDING_AND_RUNTIME_PLAN.md` for the two-lane asset strategy.
