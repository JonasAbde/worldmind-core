# Creator Mode

## Vision

WorldMind can become Roblox for AI worlds: players create agents, factions, locations, skills, laws and scenarios.

## Creator features

- prompt-to-agent.
- prompt-to-faction.
- prompt-to-location.
- scenario generator.
- story seed generator.
- permission/safety template.
- publish/share worlds.

## Prompt-to-agent output

- AGENTS.md
- SKILLS.md
- GOALS.md
- MEMORY_SEED.md
- TOOLS.json
- PERMISSIONS.json
- RELATIONSHIPS.json

## Guardrails

Generated agents cannot receive admin/world-changing permissions by default.
