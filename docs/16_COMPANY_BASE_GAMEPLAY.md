# Company / Base Gameplay

## Purpose

Founder mode gives long-term progression.

## Player base path

```txt
Apartment → small office → workshop → agent lab → company HQ → district infrastructure
```

## Company mechanics later

- start company.
- hire agents.
- assign roles.
- pay salaries.
- take contracts.
- buy equipment.
- upgrade server room.
- create agent lab.
- compete with agent companies.

## First company loop

Player creates a delivery workflow after The Missing Delivery. This can become the first founder-mode unlock.
