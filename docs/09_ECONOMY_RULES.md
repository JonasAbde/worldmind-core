# Economy Rules v0.1

## Purpose

Economy should create pressure and story, not spreadsheet complexity.

## MVP resources

- food
- tools/parts
- information
- labor
- reputation
- agent modules

## v0.1 loop

Sara stock decreases over time. If stock drops below threshold:

```txt
stock low → food scarcity → food price index rises → Sara stress rises → revenue falls → incident pressure increases
```

## Economy affects

- trade prices.
- task rewards.
- agent stress.
- theft/sabotage risk later.
- faction opportunity later.

## Later economy

- supply/demand per district.
- company payroll.
- rent/base costs.
- black-market pricing.
- faction-controlled supply lines.
