# MVP Scope v0.1

## Type

Text/2D simulation dashboard. Ikke 3D endnu.

## Goal

Bevis at 10 agents i 4 locations kan skabe memory, relationer, rumors, economy pressure og emergent incidents.

## Included

- 10 NPCs + player.
- 4 locations.
- Memory, relationship, rumor, economy, action/tool, incident og Leno summary.
- Static dashboard.
- Eval/test suite.

## Excluded for v0.1

- 3D movement.
- Combat.
- Multiplayer.
- Full creator mode.
- Voice.
- Real-world connectors.
- Marketplace.

## Success criteria

Se `25_EVAL_TEST_SUITE.md`.
