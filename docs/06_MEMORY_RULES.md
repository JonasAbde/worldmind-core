# Memory Rules v0.1

## Principle

Memory is not truth. Memory is agent interpretation.

```txt
World Event = what happened
Memory = agent interpretation
Rumor = social spread
Relationship = emotional consequence
```

## Memory types

- episodic
- relationship
- secret
- public
- rumor
- goal

## Create memory when

- relationship changes meaningfully.
- agent receives help.
- betrayal/threat occurs.
- important rumor is heard.
- evidence is discovered.
- task succeeds/fails.
- major economic loss/gain occurs.
- event importance >= 3.

## Do not create memory for

- trivial movement.
- repeated smalltalk.
- tiny trades.
- duplicate low-importance events.

## Visibility

- private
- shared
- public
- faction
- secret

Secret memories must never be shown to Leno/player without evidence.
