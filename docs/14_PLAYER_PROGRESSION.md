# Player Progression

Progression should be influence-based, not only XP.

## Progression tracks

- Social trust network.
- Investigation skill.
- Technical skill.
- Business/founder capability.
- Faction reputation.
- Leno evolution.
- Base/company upgrades.
- World knowledge.

## Stages

```txt
Early: talk, inspect, ask Leno.
Mid: hire agents, start small company, influence factions.
Late: shape governance, create agents, control infrastructure.
Endgame: city state outcome.
```

## Leno progression

- Level 1: explains world.
- Level 2: tracks rumors.
- Level 3: suggests strategies.
- Level 4: drafts deals.
- Level 5: tool access with approval.
- Level 6: opinions.
- Level 7: disagreement.
- Level 8: partner/rebel/leader path.
