# WorldMind — Game Design Summary

WorldMind er et living open-world AI simulation game, hvor hver NPC er en sandboxed agent med memory, goals, skills, tools, relationships, economy og permissions. Spillet starter i **New Aarhus District 01**, en nordic near-future/cozy cyberpunk bydel med små virksomheder, AI-tech, myndighedskontrol og underground agent-marked.

## Player fantasy

Spilleren starter som citizen/technician/founder og udvikler sig mod:

```txt
Citizen → Founder → World Shaper
```

Spilleren er ikke “chosen one”. Spilleren er en forstyrrende kraft i en levende simulation.

## Core loop

1. Spilleren observerer byen.
2. Agents lever deres egne liv.
3. Simulation skaber problemer.
4. Spilleren opdager konflikter gennem dialogue, rumors, economy og Leno.
5. Spilleren handler.
6. Agents husker.
7. Relationer og world state ændrer sig.
8. Nye incidents/quests opstår.

## Story principle

Ikke: writer skriver 100 quests.

Men: simulation skaber situationer, som Story/GM layer fortolker som quests.
